# Projetos React - Formação React Developer (DIO)

Este repositório reúne os projetos desenvolvidos durante a **Formação React Developer** da Digital Innovation One (DIO).  
O objetivo é apresentar de forma clara e objetiva os conceitos, práticas e desafios absorvidos ao longo da trilha de aprendizado, evidenciando a evolução prática com React.

---

## 🎯 Objetivo

Consolidar os conhecimentos adquiridos na formação, aplicando-os em projetos práticos que exploram:
- Componentização
- Hooks
- Gerenciamento de estado
- Navegação entre páginas
- Integração com APIs
- Validação de formulários
- Boas práticas de desenvolvimento em React

---

## 📂 Estrutura do Repositório

O repositório está organizado em diretórios, cada um representando um projeto ou desafio específico da formação:

- **hookforms**  
  Complementação do desafio de formulário de login, com:
  - Botão de login dinâmico habilitado apenas após validação dos campos.
  - Regras de validação aplicadas com React Hook Form.
  - Alerta exibido ao efetuar login com sucesso.  
  Baseado no desafio: https://github.com/digitalinnovationone/trilha-react-desafio-4.git

- **dio-replica**  
  Projeto mais completo do repositório, abrangendo praticamente todos os conceitos de um bom projeto **fullstack**.  
  Principais características:
  - Navegação entre páginas utilizando **React Router**.
  - Integração com APIs para consumo de dados dinâmicos.
  - Validação de formulários com regras robustas.
  - Estrutura organizada e escalável, refletindo boas práticas de desenvolvimento.
  - Demonstração prática de como unir front-end e back-end em um fluxo integrado.  

- *(Outros projetos podem ser adicionados e descritos conforme forem concluídos, cada um refletindo um módulo ou prática da formação.)*

## ▶️ Como Executar Localmente

1. Clone o repositório:
   git clone https://github.com/GabrielVithor99/projetos-react.git

2. Acesse o diretório do projeto desejado:
   cd projetos-react/<nome-do-projeto>

3. Instale as dependências:
   npm install

4. Execute o projeto:
   npm start

5. O projeto estará disponível em:
   http://localhost:3000
